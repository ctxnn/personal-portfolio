# ✦ Chirag Taneja — Personal Portfolio

Building intelligence from first principles. My personal slice of the internet, showcasing my work in Deep Learning, AI Research, and software engineering.

**[🌐 View Live Site](https://ctxnn.github.io/personal-portfolio/)**

## ⚡ Features

- **Luminous Aesthetics:** Premium dark-themed design inspired by the clean, minimal aesthetic of modern developer portfolios, complete with a seamless Light Mode toggle.
- **Interactive Experiences:** Custom interactive particle system (`canvas`) that reacts to mouse movement, and a custom circular planet cursor with a Saturn-like ring that magnetically scales on hover.
- **Micro-interactions:** Smooth scroll-reveal animations, magnetic project cards, and a parallax orb hero section.
- **Fully Responsive:** Fluid typography and grid systems that scale elegantly across all devices.
- **Zero Dependencies:** Built entirely from scratch using pure HTML, Vanilla CSS, and Vanilla JavaScript. No frameworks, no bloat.

## 🛠️ Tech Stack

- **HTML5** (Semantic structure)
- **CSS3** (CSS Variables, Glassmorphism, CSS Grid/Flexbox, Custom Keyframe Animations)
- **Vanilla JavaScript** (ES6+, IntersectionObserver, HTML5 Canvas API)

## 📁 Repository Structure

```text
.
├── index.html      # Main markup document (hero, about, projects, skills, timeline)
├── styles.css      # Styling, layout, animations, and dark/light mode variables
└── main.js         # Interactive canvas, scroll reveals, cursor logic, theme toggle
```

## 🚀 Local Development

1. Clone the repository:
```bash
git clone https://github.com/ctxnn/personal-portfolio.git
```
2. Open `index.html` in your favorite browser, or serve it locally:
```bash
# Using python
python3 -m http.server

# Or using Node
npx serve .
```
